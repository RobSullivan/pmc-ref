About API

PMC reference checker is also an api.

Articles

The articles collection accepts a doi as a parameter

For example, /api/v1/articles/doi/10.1038%2Fnature10158 will return all references for this article

Give it a try: /api/v1/articles/doi/10.1038%2Fnature10158

To return just the references that are free append the query references=free.

For example: /api/v1/articles/doi/10.1038%2Fnature10158?references=free

The status of each article can be found by adding "&count=percent".

For example: /api/v1/articles/doi/10.1038%2Fnature10158?references=free&count=percent

Journals

The journals collection accepts a journal title as a parameter. Journal titles are stored using their ISO abbreviation. A list of journals and their ISO abbreviations is hosted here by ncbi.nih.gov in various formats. Again, a list of a few journal titles can be found below to get started.

Please note there isn't any pagination on this call at the moment and some journals have up to 2000 articles.

To return the articles for a particular journal use /api/v1/journals/title/:title.

Give it a try: /api/v1/journals/title/Nature
