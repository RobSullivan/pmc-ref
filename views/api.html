About API

Articles

The articles collection accepts a doi as a parameter

For example, /api/v1/articles/doi/10.1038%2Fnature10158 will return all references for this article.

To return just the references that are free append the query references=free.

/api/v1/articles/doi/10.1038%2Fnature10158?references=free

The status of each article can be found by adding "&count=percent".

/api/v1/articles/doi/10.1038%2Fnature10158?references=free&count=percent

Journals

The journals collection accepts a journal title as a parameter.

/api/v1/journals/title/Nature

Other journals include:

  • Phil. Trans. R. Soc. Lond. B
  • Curr Psychiatry Rev
  • J Mov Disord
  • Biom J
  • OA Autism
  • Nature
  • Science
  • Mech. Dev.
  • AATCC Rev
