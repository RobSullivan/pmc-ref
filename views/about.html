About API

PMC reference checker is a tool to quickly find what percentage of an article's references are free to access in PubMed Central and also return a list of all those references. The idea is if you need to buy an article you can see which of it's references you may also need to pay for and which ones you don't.

A few dois to get started:

  • 10.1371/journal.pgen.1004432
  • 10.1038/nbt.1667
  • 10.1038/nature11543
  • 10.1016/j.imlet.2006.09.005
  • 10.1038/hdy.2010.93
  • 10.3389/fpls.2014.00367
  • 10.1371/journal.pone.0088397
  • 10.1186/1471-2164-14-741
  • 10.1007/s10719-013-9486-6
